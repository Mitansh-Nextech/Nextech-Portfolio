Choose Your Own Adventure Story

This is a project that was assigned to me in my intership. In this project, I had to create a choose your own adventure story. So I decied me make mine about an hiring manager who is hiring a new employee. The story starts with the hiring manager looking at the candidate's traits and then the user has to make choices that will affect the outcome of the story. The story has multiple endings and the user can choose to go back and make different choices to see how the story changes. This project was made by using HTML, CSS, and Java Script, which was used to change the DOM of the html file and change the style of the CSS file.
